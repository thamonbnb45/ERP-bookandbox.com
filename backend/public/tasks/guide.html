📋 คู่มือใช้งาน Task Tracker V3

คู่มือฉบับย่อ — กดเข้าใช้ที่ /tasks

วิธีใช้งานหลัก

🖱️ ลากการ์ดเปลี่ยนสถานะ
กดค้างที่การ์ด → ลากไปคอลัมน์ที่ต้องการ → ปล่อย = สถานะเปลี่ยนทันที + บันทึกเวลาอัตโนมัติ
📋 ดูรายละเอียด + แก้ไข
กดที่การ์ด → เปิดแผงด้านขวา → แก้ชื่อ/คน/วัน/สถานะได้เลย (กดข้างนอก = บันทึก)
☑️ Checklist
ในแผงรายละเอียด → พิมพ์รายการ → กด + → ติ๊กเมื่อเสร็จ → ดู % ที่แถบสี
💬 Comments
แท็บ Comments → เลือก emoji ตัวเอง → พิมพ์ → กดส่ง
📊 Activity Log
แท็บ Activity → เห็นทุกการเปลี่ยนแปลง (ใคร ทำอะไร เมื่อไหร่)
👤 กรองงานรายคน
กดปุ่ม emoji ที่แถบบน → เห็นเฉพาะงานของคนนั้น
📅 Calendar
กดปุ่ม Calendar → ดูงานตามวันกำหนดส่ง → ◀▶ เลื่อนเดือน
📄 Export CSV
กดปุ่ม CSV → ดาวน์โหลด → เปิดใน Google Sheets / Excel

📱 สั่งงานผ่าน LINE

💡 ไม่ต้องเปิดเว็บ พิมพ์ใน LINE กลุ่มได้เลย!
เสร็จ #1 → งาน ID 1 เป็น ✅
ติด #2 เครื่องเสีย → งาน ID 2 เป็น 🚨 + บันทึกเหตุผล
เริ่ม #3 → งาน ID 3 เป็น ⚙️

🔔 แจ้งเตือนอัตโนมัติ

8:00 / 12:00 / 16:00 — แจ้งงานค้างเกิน 24 ชม.
9:30 — Pre-Meeting Report สรุปงานประจำวัน

❓ คำถามที่พบบ่อย

ลากไม่ได้บนมือถือ? → กดค้าง 0.5 วินาที แล้วค่อยลาก
แก้ชื่อแล้วไม่บันทึก? → กดข้างนอกช่อง (blur) ถึงจะบันทึก
ดู Task ID? → กดเข้าการ์ด → เห็น ID: #1 ด้านบน
